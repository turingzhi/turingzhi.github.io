    some frequently used commands

    cat & less & tail & vim

    • less filename.log
      • / search_word
    • tail -f filename.log
      • f follow
    • vim filename.log

    ping & telnet & traceroute & dig

    • ping http://www.xxx.com
      • check the network connection
    • telnet www.xxx.com port_num
      • check the port connection

    netstat

    • netstat -lntp
      • check the port status
      • l listening,netstat -l shows the listening sockets
      • n number
      • t tcp
      • p process
    • netstat -latp
      • l listening,netstat -l shows the listening sockets
      • a all
      • t tcp
      • p process

    ps

    • ps -ef
      • ps process status
      • e select all processes
      • f full format

    top & df & du

    • top
      • check the cpu and memory usage
    • df -h
      • check the disk usage
      • h human-readable
    • du -sh
      • check the disk usage

    > & >> & < & |

    • bash xxx.sh > xxx.log
      • > overwrite
    • bash xx.sh >> xxx.log
      • >> append
      • 2>&1 redirect the error message to the standard output
    • bash xxx.sh < xxx.log
      • < input
      • | pipe

      • echo $$ print the process id
      • echo $! print the last process id
